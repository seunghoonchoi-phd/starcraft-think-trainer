앱은 첫 100초 동안 입력 과제와 판단 과제를 따로 측정합니다. 이후 사용자가 두 과제를 함께 수행하면, 앱은 입력 속도와 판단 정확도가 각각 얼마나 달라졌는지 계산합니다. 앱은 한 번에 한 조건만 바꿔 난이도를 조절합니다.

앱이 원 안에 숫자를 표시하면, 사용자는 같은 숫자 키를 누른 뒤 원을 클릭합니다. 앱이 판단 카드를 표시하면, 사용자는 Q, W, E 중 정답에 해당하는 키를 누릅니다.

이 앱은 키보드와 마우스를 사용하는 데스크톱 환경을 지원합니다. 이 앱은 사용자의 건강 상태나 지능을 검사하지 않습니다.

    이 기기에 저장한 최근 기록

    사용자는 앱 기록과 실제 게임 기록을
    따로 비교해야 합니다

    앱 기록은 이 훈련 과제에서 측정한 입력 속도와 판단 정확도를 보여 줍니다. 사용자는 실제 게임 리플레이를 열어 교전 중 생산 공백 시간, 보급 한도 초과 시간, 주요 판단 누락 횟수가 줄었는지 따로 확인해야 합니다.

    사용자가 첫 세션을 끝내면 앱이 최근 기록을 여기에 표시합니다.

    훈련 방법

    사용자가 입력과 판단을 함께 수행하면
    사용자의 반응 시간이 길어질 수 있습니다

    사용자가 짧은 시간에 두 행동을 연속으로 선택하면, 두 번째 행동의 반응 시간이 길어질 수 있습니다. 이 앱은 사용자가 고정된 입력 순서를 연습한 뒤 입력 과제와 판단 과제를 함께 수행하게 합니다. 앱은 두 과제의 정확도가 기준을 넘을 때만 표적 표시 간격을 줄입니다.

    01

    두 과제를 따로 측정

    사용자는 입력 과제와 판단 과제를 따로 수행합니다. 앱은 각 과제의 속도와 정확도를 기준선으로 저장합니다.

    02

    고정 입력 순서 연습

    사용자는 1→2→3→4 순서로 숫자 키와 클릭을 반복합니다. 앱은 틀리거나 반복해서 누른 키를 점수로 계산하지 않습니다.

    03

    입력하면서 판단

    사용자는 입력 순서를 계속 수행하면서 우선순위 문제를 풉니다. 사용자가 한 과제를 중단하면 앱은 해당 구간을 통과로 처리하지 않습니다.

    04

    바뀐 조건에서 검사

    앱은 입력 순서와 판단 규칙을 바꾼 뒤 정답을 바로 알려 주지 않습니다. 이 검사는 사용자가 연습과 다른 문제에도 규칙을 적용하는지 측정합니다.

    연구 근거와 한계

    이 앱은 연구자들이 확인하지 않은 효과를
    주장하지 않습니다

    연구자들은 참가자에게 입력과 판단을 함께 수행하게 했습니다. 연구자들은 반복 연습, 과제 난이도, 연습 순서가 참가자의 수행 결과를 어떻게 바꾸는지도 조사했습니다. 이 앱은 그 연구 결과를 참고해 훈련 순서를 정했습니다. 연구자들은 이 앱이 스타크래프트 래더 티어를 올리는지 직접 시험하지 않았습니다.

    이 앱이 제공하는 기능
    “이 앱은 입력만 할 때와 입력하면서 판단할 때의 속도와 정확도를 비교합니다. 사용자는 제한된 시간 안에 입력 순서를 수행하고 우선순위 문제를 풉니다.”

    이 앱은 뇌 기능 향상, ADHD 증상 개선, 스타크래프트 실력 상승을 보장하지 않습니다.
